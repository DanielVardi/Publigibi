Editor de Quadrinhos

Crie suas próprias histórias em quadrinhos com nosso editor intuitivo. Desenhe molduras, adicione imagens e exporte seu trabalho para compartilhar com o mundo!

Como usar o editor:

  • Criar molduras: Clique na tela para criar os pontos da sua moldura. Clique perto do primeiro ponto para fechar a forma.
  • Adicionar imagens: Arraste e solte uma imagem dentro de uma moldura para adicioná-la.
  • Mover imagem: Selecione uma moldura com imagem e arraste para mover o conteúdo.
  • Ajustar zoom: Use os botões de zoom para ampliar ou reduzir a imagem dentro da moldura.
  • Selecionar molduras: Clique em uma moldura para selecioná-la (borda laranja).
  • Remover molduras: Clique com o botão direito + Ctrl em uma moldura para removê-la.
  • Exportar: Use os botões de exportação para salvar seu trabalho como JPG ou PDF.
